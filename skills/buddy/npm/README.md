# Buddy Skill Creator — npm Installer

Distill your ideal buddy into an AI Skill. Import chat history, photos, social media posts, or just describe your dream buddy — generate Vibe Memory + Persona with continuous evolution.

Powered by [EvoLink API](https://evolink.ai?utm_source=npm&utm_medium=skill&utm_campaign=buddy)

## Install

```bash
npx evolinkai-buddy-skill
```

## What This Does

Installs the Buddy Skill Creator into your OpenClaw/ClawHub skills directory. After installation, use `/create-buddy` in Claude Code to start creating your AI buddy.

## Features

- Multiple data sources: WeChat, QQ, screenshots, photos, or pure imagination
- Buddy types: lunch buddy, study buddy, gaming buddy, gym buddy, and more
- Two-layer architecture: Vibe Memory + Persona
- Evolution: add memories, correct responses, version history
- AI analysis via EvoLink API (Claude models)
