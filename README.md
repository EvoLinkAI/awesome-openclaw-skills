This repository is a curated **OpenClaw skills collection** built for real-world AI workflows. It is designed as a central hub for the best **OpenClaw skills** across content creation, automation, web research, developer tooling, productivity, communication, and media generation.

If you are looking for **OpenClaw skills for automation**, **OpenClaw skills for research**, **OpenClaw skills for content creation**, or **OpenClaw skills for business workflows**, this repository is the place to start.

## FAQ

### What are OpenClaw skills?

OpenClaw skills are reusable capability packages that extend what an OpenClaw agent can do. A skill can add workflow logic, external integrations, scripts, prompts, setup guidance, or task-specific behaviors.

### What are the best OpenClaw skills for real-world use?

It depends on the workflow. Popular skill categories include AI video generation, web research, PDF processing, Gmail automation, Google Sheets workflows, translation, SEO support, and coding assistance.

### Can I use these OpenClaw skills for business workflows?

Yes. Many of these skills are intended for practical use cases such as research, communication, reporting, content operations, automation, and media production.

### Will each skill keep its own documentation?

Yes. The plan is for every skill to keep its own folder, `SKILL.md`, and README so each skill can still rank for its own long-tail search terms and remain easy to install.

### Why merge many skill repos into one repository?

A single OpenClaw skills repository makes discovery easier, improves maintenance, creates stronger internal linking, and gives the overall project a better SEO footprint.
