# Migration Plan

This repository is the consolidation target for multiple standalone OpenClaw skill repositories.

## Migration Principles

1. Preserve each skill as its own folder.
2. Preserve each skill's `SKILL.md` and unique usage docs.
3. Keep the root README focused on discovery, SEO, and navigation.
4. Use skill-level README files to target long-tail search queries.
5. After consolidation is complete, rename the repository to reflect the broader multi-skill scope.
