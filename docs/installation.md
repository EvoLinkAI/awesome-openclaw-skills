# OpenClaw Skills Installation Guide

This repository is being consolidated into a multi-skill OpenClaw hub.

## Install from GitHub

```bash
openclaw skills add https://github.com/EvoLinkAI/seedance2-video-gen-skill-for-openclaw
```

## Planned Multi-Skill Layout

Each skill will live under `skills/<name>/` with its own:

- `SKILL.md`
- `README.md`
- optional `package.json`
- supporting scripts, references, and assets

## Goals

- make OpenClaw skills easier to discover
- improve skill installation clarity
- support SEO for both the main repository and individual skill pages
- simplify long-term maintenance across many skill packages
