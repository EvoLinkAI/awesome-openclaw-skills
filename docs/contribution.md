# Contribution Guide

## Scope

This repository is evolving from a standalone skill repo into a central OpenClaw skills collection.

## Contribution Rules

- keep each skill self-contained in `skills/<name>/`
- preserve installability and documentation clarity
- avoid mixing unrelated scripts at the root
- keep README updates aligned with SEO and navigation goals
- document migration changes clearly when moving content from older repos

## Documentation Standards

- root README targets broad discovery keywords and category navigation
- each skill README should target its own use case and long-tail keywords
- preserve clean internal links between root docs and skill docs
